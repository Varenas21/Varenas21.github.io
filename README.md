## Portfolio Description
Welcome to my online portfolio using GitHub Pages! This portfolio showcases my work as a Software Engineer/Developer and Game Developer. It highlights my skills, projects, and experiences in full stack development, game design, UX/UI design and more! The portfolio is designed to give potential clients, employers, and collaborators a clear understanding of my capabilities and projects I've worked on.

### Features
- **Responsive Design**: The site is fully responsive, ensuring a smooth experience across all devices.
- **About Me Section**: A brief introduction to my background, skill, and what drives me in my professional journey.
- **Project Showcase**: A collection of my best work, each with detailed descriptions, technologies used, and links to Github repositories and release pages.
- **Contact Information**: Easy access to my contact details, allowing visitors to reach out for collaborations or job opportunities.

### Technologies Used:
- **Frontend**: HTML, CSS, Javascript
- **Backend**:Node.js
- **Styling**:TailwindCSS (RETIRED)
- **Other**: Visual Studio Code, GitHub
