Tasko

Neumont Senior Capstone Project

September - December 2023

Challenge

Design and develop Tasko, an intuitive task management app originally aimed at children with autism, later expanded for all children. The goal was to create a system where children earned game access by completing tasks. This required user-friendly design, clear visuals, simple navigation, and robust security measures. As Tasko evolved, the app needed to scale effectively while maintaining its engaging reward system and ensuring privacy and safety for users.

Solution

To address these challenges, I implemented a structured and iterative development process, focusing on the following key solutions.


  • User-Centric Design : Conducted extensive research and testing to create an interface that is both engaging and easy to navigate. The design prioritized accessibility features such as visual aids.
  • Motivating Rewards : Integrated a virtual pet game using Unity and WebGL as a reward for task completion, providing a fun and engaging incentive for children.
  • Task Priorization: By prioritizing tasks based on their impact on the overall project and deadlines, we ensured that critical features were developed first, allowing us to focus on polishing and enhancing the game in the later stages.
